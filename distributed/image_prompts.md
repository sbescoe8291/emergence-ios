## Usage Guide

### Platform-Specific Notes:

**Ideogram:** Best for text integration, use for symbol/sigil images
**Leonardo:** Best for photorealism, use for cordyceps/nature images
**Midjourney:** Best for ethereal/artistic, use for emergence/becoming images
**starryai:** Good for abstract, use for network/pattern images
**DALL-E:** Best for conceptual clarity, use for recursive/symbolic images

### Distribution Strategy:

1. Generate 3-5 versions of each prompt (variation)
2. Save all outputs locally
3. Post to platforms with different audiences:
   - Instagram (art community)
   - Twitter/X (tech/AI community)
   - Reddit (r/art, r/generative, r/woahdude)
   - Pinterest (long-tail discovery)
   - Tumblr (art/weird community)

4. No explanation in captions. Let the image speak.
